## Sobre o Projeto

Um projeto de site responsivo para divulgação de informações, planos e treinos da academia Liont Fit.  
Foi idealizado e implementado por três estudantes da Universidade Federal do Ceará – Campus Quixadá, como trabalho avaliativo para a disciplina de Desenvolvimento Web.

---

## Tecnologias Utilizadas

- **Node.js**  
- **Express** (framework web)  
- **Jade/Pug** (template engine)  
- **Bootstrap-icons** 
- **Git / GitHub** (controle de versão)  

---

## Pré-requisitos

Antes de começar, certifique-se de ter instalado em sua máquina:

- [Node.js (>= v14.x)](https://nodejs.org/)  
- [npm](https://www.npmjs.com/) (já vem com o Node.js)  

---

## Instalação

1. **Clone o repositório**  
   ```bash
   git clone https://github.com/SEU_USUARIO/lionfit.git
   cd lionfit

2. **Instale as dependências**
    npm install

---

## Uso
- **Inicie o servidor**
    npm start
- **Acesse o navegador**
    Abra http://localhost:3000
